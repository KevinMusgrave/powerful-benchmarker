    config_models¶

    models¶

    An object mapping from strings to the models that create embeddings.

    Default yaml:

    models:
  trunk:
    bninception:
      pretrained: imagenet
  embedder:
    MLP:
      layer_sizes:
        - 128

    Command line:

    --models {trunk: {bninception: {pretrained: imagenet}}, embedder: {MLP: {layer_size: [128]}}}
